# Kipawa React Dashboard

## Overview
Admin dashboard for managing farms, cooperatives, and compliance.

## Tech Stack
- React 18 with TypeScript
- React Router v6
- Leaflet for map visualization
- TanStack Query for data fetching
- TailwindCSS for styling

## Getting Started
```bash
npm install
npm start
```

## Available Scripts
- `npm start` - Development server
- `npm test` - Run tests
- `npm run build` - Production build
- `npm run lint` - Lint code
