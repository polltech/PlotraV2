Introduction to Plotra

Plotra is a comprehensive digital platform designed specifically for coffee farmers and cooperatives to manage their operations, ensure regulatory compliance, and build sustainable agricultural practices.

Key Features

  • Farm Mapping: GPS-enabled parcel mapping and boundary definition
  • Production Tracking: Monitor yields, quality, and delivery records
  • EUDR Compliance: Automated due diligence statement generation
  • Cooperative Management: Multi-user access and data sharing
  • Real-time Analytics: Insights for better decision making

Who is Plotra for?

Plotra serves three main user types:

  • Farmers: Individual coffee growers managing their land and production
  • Cooperatives: Organizations managing multiple farmers and collective operations
  • Administrators: System administrators overseeing platform operations

Getting Started

Creating Your Account

1
Visit the Platform

Go to the Plotra dashboard and click "Join Our Community" on the landing page.

2
Choose Your Role

Select whether you're a farmer, cooperative officer, or administrator.

3
Provide Information

Fill in your personal details, contact information, and organizational affiliation.

4
Verify Email

Check your email for a verification link to activate your account.

Dashboard Overview

Once logged in, you'll see the main dashboard with:

  • Quick Stats: Overview of your farms, parcels, and recent activity
  • Map View: Visual representation of your farm locations
  • Navigation Menu: Access to all platform features
  • Notifications: Important updates and reminders

Farm Mapping

Accurate farm mapping is crucial for compliance and effective farm management.

Creating a New Farm

  1. Navigate to "My Farms" in the sidebar
  2. Click "Add New Farm"
  3. Enter farm details (name, location, size)
  4. Use the map interface to define boundaries
  5. Save your farm

Mapping Methods

Method Description Best For Accuracy
GPS Drawing Draw boundaries directly on the map Large areas High
Walking Mode Walk the perimeter with GPS tracking Precise boundaries Very High
Import KML Upload existing boundary files Existing data Variable

API Reference

The Plotra API allows developers to integrate with our platform programmatically.

Authentication

All API requests require authentication using Bearer tokens.

Authorization: Bearer YOUR_API_TOKEN

Endpoints

GET /api/v2/auth/me

Get current user information

GET /api/v2/farmer/farm

Retrieve user's farm information

POST /api/v2/farmer/farm/{farm_id}/parcels

Create a new parcel within a farm

POST /api/v2/admin/eudr/dds

Generate EUDR Due Diligence Statement

Troubleshooting

Common Issues

GPS Location Not Working

Solution: Ensure location services are enabled in your browser and device settings. Try refreshing the page.

Map Not Loading

Solution: Check your internet connection. Clear browser cache and try again. If the issue persists, contact support.

Cannot Save Farm Data

Solution: Ensure all required fields are filled. Check your internet connection and try again.
